\# Python File Organizer



A Python automation tool that automatically organizes files in a folder based on their file types.



\## Features

\- Organizes files into categories:

&nbsp; - Images

&nbsp; - Documents

&nbsp; - Videos

&nbsp; - Music

&nbsp; - Archives

&nbsp; - Others (uncategorized)

\- Automatically creates folders if they don't exist

\- Cleans messy folders with one command
