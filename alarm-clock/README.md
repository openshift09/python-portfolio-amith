\# Python Alarm Clock



A simple Python-based alarm clock that rings at a specified time.



\## Features

\- Accepts alarm time in HH:MM:SS format

\- Continuously checks current time

\- Plays sound when alarm time is reached

\- Uses the `playsound` module
