\# Python Email Sender



A Python script to send emails using Gmail SMTP and App Password authentication.



\## Features

\- Sends plain text emails

\- Uses secure Gmail SMTP with TLS

\- User provides sender, receiver, subject, body

\- Real-world automation skill



\## Requirements

Enable 2-Step Verification in Gmail  

Create a Gmail App Password  

https://myaccount.google.com/apppasswords
