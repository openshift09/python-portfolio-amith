 To-Do List Application (Python)

A simple command-line To-Do List application using Python.  
Tasks are stored in a text file (`tasks.txt`) to preserve them even after the program is closed.

## Features
- Add new tasks
- View all tasks
- Delete tasks
- Persistent storage (tasks are saved in a file)
- User-friendly menu
