\# Python Expense Tracker



A simple Expense Tracker that stores expenses in a CSV file and allows users to:



\- Add expenses

\- View all expenses

\- Calculate total spending

\- Store entries with date, category, amount, and description



\## Features

\- CSV-based storage

\- Auto timestamps

\- Clean CLI interface

\- Total expense calculation
