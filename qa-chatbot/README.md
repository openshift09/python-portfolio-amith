\# Python Q\&A Chatbot (JSON Knowledge Base)



A smarter chatbot that uses a JSON file as a knowledge base.  

It searches for keywords and responds with predefined answers.



\## Features

\- Loads answers from a JSON file

\- Supports multiple topics (Python, DevOps, Docker, Kubernetes, etc.)

\- Easy to expand knowledge base

\- Much smarter than rule-based chatbot
