\# Python Password Strength Checker



A Python program that checks how strong a password is based on multiple criteria.



\## Features

\- Checks for:

&nbsp; - Minimum length (>= 8 chars)

&nbsp; - Lowercase letters

&nbsp; - Uppercase letters

&nbsp; - Numbers

&nbsp; - Special characters

\- Classifies password as Strong, Medium, or Weak
